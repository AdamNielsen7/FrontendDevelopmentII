Week 8 Notes

  • Transform Functions
    • translate moves element relative to its current position 
                      transform: translate(45px, -45px);
    • Scale scales element horizontally then vertically (add multiple transforms) 
                      transform: translateX(40px) scale(1.5);
    • rotate(deg)
    • skew(x,y)
    • transform-origin: 0 0; lets you set the origin of the transformation
  • Transition Functions
    • These are the same
                      .ad-ad2 h1 span {
                    transition-property: transform; 
                    transition-duration: 0.2s;  
                    transition-timing-function: ease-out;
                    transition-delay: 50ms;
                }
                .ad-ad2 h1 span {
                    transition: transform 0.2s ease-out 50ms;
                }
  • Animations
    • animations use keyframes 
                      @keyframes appearDisappear { 
                    0%, 100% {
                    opacity: 0;
                    }
                    20%, 80% {
                    opacity: 1;
                    }
                }
    • verbose and concise 
                      .verbose {
                    animation-name: appearDisappear; 
                    animation-duration: 300ms; 
                    animation-timing-function: ease-in;
                    animation-iteration-count: 1;
                    animation-direction: alternate;
                    animation-delay: 5s;
                    animation-fill-mode: backwards;
                    animation-play-state: running;
                }
                /* shorthand */
                .concise {
                    animation: 300ms ease-in alternate 5s backwards appearDisappear;
                }

    • "draw" a square with a blue fill 
                      var canvas = document.getElementById("myCanvas"); 
                var context = canvas.getContext("2d"); 
                context.strokeStyle = "red";
                context.fillStyle = "rgba(0, 0, 255, 0.5)";
                context.fillRect(10, 10, 100, 100);   
                context.strokeRect(10, 10, 100, 100);
    • canvas coordinate system
    • write a function to draw a circle from an arc 
                      function drawCircle(canvas) {
                    var canvas = document.getElementById("myCanvas");
                    var context = canvas.getContext("2d");
                    context.beginPath();
                    context.arc(50, 50, 30, 0, Math.PI*2, true);
                }
